# Animated Quiz Chart Example

This is an example app demonstrating how to use the `animated_quiz_chart` package.

## Features Demonstrated

1. **Standalone Chart**: Shows how to use `AnimatedQuizChart` widget
2. **Full Result Page**: Shows how to use `ResultPage` widget
3. **Real-time Customization**: Interactive controls to modify chart properties
4. **Code Examples**: Shows actual usage code
